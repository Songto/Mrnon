# Badge art (animated)

Drop one animated **GIF** per advanced badge here, named by its id. Until a
file exists, the built-in SVG medallion shows automatically.

- Format: animated GIF (transparent background)
- Size: 256×256 px square, keep each file under ~500 KB

Filenames (5 advanced badges):
- ourchat.gif — completed all quests
- secret.gif — special person (admin-granted)
- gardener.gif — collected all seeds
- cutefactor.gif — cute person (admin-granted)
- famous.gif — 500+ profile likes
