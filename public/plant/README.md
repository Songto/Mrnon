# Pot / plant art (animated)

Drop one animated **GIF** per growth stage here. Until a file exists, the
built-in SVG plant shows automatically.

- Format: animated GIF (transparent background)
- Size: 512×512 px square, keep each file under ~1 MB

Filenames (6 growth stages):
- stage-0.gif — seed / just planted
- stage-1.gif — sprout
- stage-2.gif — leafy
- stage-3.gif — budding
- stage-4.gif — bloom
- stage-5.gif — full bloom / flourishing
