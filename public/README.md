# Logo

Drop your circular logo image here named **`logo.png`** (any square image works —
it's cropped to a circle in the nav). Recommended: 256×256 or larger PNG.

- Nav/header logo: `public/logo.png`
- Browser-tab favicon also uses `public/logo.png`

Until `logo.png` exists, the header falls back to the 🍓 emoji.
