# Ambience sounds

The ambience mixer (bottom-right 🔈) has 6 sounds:

**Built-in (no file needed):** Rain, Fireplace, Chimes — synthesized live.

**Your custom loops — drop files here:**
- `lofi.mp3`   → "Lofi" 🎶
- `cafe.mp3`   → "Café" ☕
- `forest.mp3` → "Forest" 🌳

Tips:
- Use **seamless looping** clips (they repeat). `.mp3` or `.ogg`, ideally a
  few minutes long and < ~3 MB each.
- Rename/relabel slots or add different ones by editing the `SOUNDS` list in
  `src/components/AmbiencePlayer.tsx`.
- Until a file exists, its button shows faded; tap it and it'll tell you which
  file to add.
